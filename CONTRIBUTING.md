# Contributing Guidelines

All commit messages **must be written in English** using the **Conventional Commits** format.

### ✅ Example formats
- `feat(detector): add dotenv-based config system`
- `fix(server): handle WebSocket reconnection after timeout`
- `refactor(overlay): simplify fade animation and remove debug logs`
- `chore: update dependencies`

Use present tense and imperative style (e.g., “add”, not “added”).
